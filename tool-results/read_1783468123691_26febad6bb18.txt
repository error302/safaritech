    32→## ⚡ Quick Start
    33→
    34→### Option 1: Install the app (Recommended)
    35→
    36→The fastest way in — no clone, no terminal. [**Agency Agents**](https://agencyagents.app) is a native desktop app (macOS · Linux · Windows) that browses the whole roster and installs agents into Claude Code, Cursor, Codex, Gemini CLI, OpenCode, Qwen, and Osaurus for you, then keeps them up to date.
    37→
    38→**[⬇ Download the latest release](https://github.com/msitarzewski/agency-agents-app/releases/latest)** — or on a Mac:
    39→
    40→```bash
    41→brew install --cask msitarzewski/agency-agents/agency-agents
    42→```
    43→
    44→Prefer the command line? The script-based options below install the same agents.
    45→
    46→### Option 2: Use with Claude Code
    47→
    48→```bash
    49→# Install all agents to your Claude Code directory
    50→./scripts/install.sh --tool claude-code
    51→
    52→# Or manually copy a category if you only want one division
    53→cp engineering/*.md ~/.claude/agents/
    54→
    55→# Then activate any agent in your Claude Code sessions:
    56→# "Hey Claude, activate Frontend Developer mode and help me build a React component"
    57→```
    58→
    59→### Option 3: Use as Reference
    60→
    61→Each agent file contains:
    62→- Identity & personality traits
    63→- Core mission & workflows
    64→- Technical deliverables with code examples
    65→- Success metrics & communication style
    66→
    67→Browse the agents below and copy/adapt the ones you need!
    68→
    69→### Option 4: Use with Other Tools (GitHub Copilot, Antigravity, Gemini CLI, OpenCode, OpenClaw, Cursor, Aider, Windsurf, Kimi Code, Codex, Osaurus, Hermes, Mistral Vibe)
    70→
    71→```bash
    72→# Step 1 -- generate integration files for all supported tools
    73→./scripts/convert.sh
    74→
    75→# Step 2 -- install interactively (auto-detects what you have installed)
    76→./scripts/install.sh
    77→
    78→# Or target a specific tool directly
    79→./scripts/install.sh --tool antigravity
    80→./scripts/install.sh --tool gemini-cli
    81→./scripts/install.sh --tool opencode
    82→./scripts/install.sh --tool copilot
    83→./scripts/install.sh --tool openclaw
    84→./scripts/install.sh --tool cursor
    85→./scripts/install.sh --tool aider
    86→./scripts/install.sh --tool windsurf
    87→./scripts/install.sh --tool kimi
    88→./scripts/install.sh --tool codex
    89→./scripts/install.sh --tool osaurus
    90→./scripts/install.sh --tool hermes
    91→./scripts/install.sh --tool vibe
    92→```
    93→
    94→**Install only the teams you need** (not everyone wants every division):
    95→
    96→```bash
    97→./scripts/install.sh                                    # interactive wizard: pick tools + teams
    98→./scripts/install.sh --tool claude-code --division engineering,security
    99→./scripts/install.sh --tool cursor --agent frontend-developer,ui-designer
   100→./scripts/install.sh --list teams                       # see every team + agent count
   101→./scripts/install.sh --tool opencode --division engineering --dry-run
   102→```
   103→
   104→> **OpenCode note:** OpenCode's runtime currently registers only ~119 agents and silently drops the rest ([upstream bug](https://github.com/anomalyco/opencode/issues/27988)). Installing a subset with `--division` keeps you under that limit. The installer warns you when a selection would exceed it.

   663→## 🔌 Multi-Tool Integrations

   667→### Supported Tools
   668→
   669→- **[Claude Code](https://claude.ai/code)** — native `.md` agents, no conversion needed → `~/.claude/agents/`
   670→- **[GitHub Copilot](https://github.com/copilot)** — native `.md` agents, no conversion needed → `~/.github/agents/` + `~/.copilot/agents/`
   671→- **[Antigravity](https://github.com/google-gemini/antigravity)** — `SKILL.md` per agent → `~/.gemini/config/skills/`
   672→- **[Gemini CLI](https://github.com/google-gemini/gemini-cli)** -- `.md` agent files -> `~/.gemini/agents/`
   673→- **[OpenCode](https://opencode.ai)** — `.md` agent files → `.opencode/agents/`
   674→- **[Cursor](https://cursor.sh)** — `.mdc` rule files → `.cursor/rules/`
   675→- **[Aider](https://aider.chat)** — single `CONVENTIONS.md` → `./CONVENTIONS.md`
   676→- **[Windsurf](https://codeium.com/windsurf)** — single `.windsurfrules` → `./.windsurfrules`
   677→- **[OpenClaw](https://github.com/openclaw/openclaw)** — `SOUL.md` + `AGENTS.md` + `IDENTITY.md` per agent
   678→- **[Qwen Code](https://github.com/QwenLM/qwen-code)** — `.md` SubAgent files → `~/.qwen/agents/`
   679→- **[Kimi Code](https://github.com/MoonshotAI/kimi-cli)** — YAML agent specs → `~/.config/kimi/agents/`
   680→- **[Codex](https://developers.openai.com/codex/overview)** — TOML custom agents → `~/.codex/agents/`
   681→- **Osaurus** -- `SKILL.md` skills -> `~/.osaurus/skills/`
   682→- **[Hermes](integrations/hermes/README.md)** -- lazy-router plugin -> `~/.hermes/plugins/`

   686→### ⚡ Quick Install
   687→
   688→**Step 1 -- Generate integration files:**
   689→```bash
   690→./scripts/convert.sh
   691→# Faster (parallel, output order may vary): ./scripts/convert.sh --parallel
   692→```
   693→
   694→**Step 2 -- Install (interactive, auto-detects your tools):**
   695→```bash
   696→./scripts/install.sh
   697→# Faster (parallel, output order may vary): ./scripts/install.sh --no-interactive --parallel
   698→```
   699→
   700→The installer scans your system for installed tools, shows a checkbox UI, and lets you pick exactly what to install:
   701→
   702→```
   703→  +------------------------------------------------+
   704→  |   The Agency -- Tool Installer                 |
   705→  +------------------------------------------------+
   706→
   707→  System scan: [*] = detected on this machine
   708→
   709→  [x]  1)  [*]  Claude Code     (claude.ai/code)
   710→  [x]  2)  [*]  Copilot         (~/.github + ~/.copilot)
   711→  [x]  3)  [*]  Antigravity     (~/.gemini/antigravity)
   712→  [ ]  4)  [ ]  Gemini CLI      (~/.gemini/agents)
   713→  [ ]  5)  [ ]  OpenCode        (opencode.ai)
   714→  [ ]  6)  [ ]  OpenClaw        (~/.openclaw/agency-agents)
   715→  [x]  7)  [*]  Cursor          (.cursor/rules)
   716→  [ ]  8)  [ ]  Aider           (CONVENTIONS.md)
   717→  [ ]  9)  [ ]  Windsurf        (.windsurfrules)
   718→  [ ] 10)  [ ]  Qwen Code       (~/.qwen/agents)
   719→  [ ] 11)  [ ]  Kimi Code       (~/.config/kimi/agents)
   720→  [ ] 12)  [ ]  Codex           (~/.codex/agents)
   721→  [ ] 13)  [ ]  Osaurus         (~/.osaurus/skills)
   722→  [ ] 14)  [ ]  Hermes          (~/.hermes/plugins)
   723→
   724→  [1-14] toggle   [a] all   [n] none   [d] detected
   725→  [Enter] install   [q] quit
   726→```
   727→
   728→**Or install a specific tool directly:**
   729→```bash
   730→./scripts/install.sh --tool cursor
   731→./scripts/install.sh --tool opencode
   732→./scripts/install.sh --tool openclaw
   733→./scripts/install.sh --tool antigravity
   734→./scripts/install.sh --tool codex
   735→./scripts/install.sh --tool osaurus
   736→./scripts/install.sh --tool hermes
   737→```
   738→
   739→**Non-interactive (CI/scripts):**
   740→```bash
   741→./scripts/install.sh --no-interactive --tool all
   742→```
   743→
   744→**Faster runs (parallel)** — On multi-core machines, use `--parallel` so each tool is processed in parallel. Output order across tools is non-deterministic. Works with both interactive and non-interactive install: e.g. `./scripts/install.sh --interactive --parallel` (pick tools, then install in parallel) or `./scripts/install.sh --no-interactive --parallel`. Job count defaults to `nproc` (Linux), `sysctl -n hw.ncpu` (macOS), or 4; override with `--jobs N`.
   745→
   746→```bash
   747→./scripts/convert.sh --parallel                    # convert all tools in parallel
   748→./scripts/convert.sh --parallel --jobs 8           # cap parallel jobs
   749→./scripts/install.sh --no-interactive --parallel   # install all detected tools in parallel
   750→./scripts/install.sh --interactive --parallel      # pick tools, then install in parallel
   751→./scripts/install.sh --no-interactive --parallel --jobs 4
   752→```
