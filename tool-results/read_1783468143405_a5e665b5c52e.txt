    32→    32→## ⚡ Quick Start
    33→    33→
    34→    34→### Option 1: Install the app (Recommended)
    35→    35→
    36→    36→The fastest way in — no clone, no terminal. [**Agency Agents**](https://agencyagents.app) is a native desktop app (macOS · Linux · Windows) that browses the whole roster and installs agents into Claude Code, Cursor, Codex, Gemini CLI, OpenCode, Qwen, and Osaurus for you, then keeps them up to date.
    37→    37→
    38→    38→**[⬇ Download the latest release](https://github.com/msitarzewski/agency-agents-app/releases/latest)** — or on a Mac:
    39→    39→
    40→    40→```bash
    41→    41→brew install --cask msitarzewski/agency-agents/agency-agents
    42→    42→```
    43→    43→
    44→    44→Prefer the command line? The script-based options below install the same agents.
    45→    45→
    46→    46→### Option 2: Use with Claude Code
    47→    47→
    48→    48→```bash
    49→    49→# Install all agents to your Claude Code directory
    50→    50→./scripts/install.sh --tool claude-code
    51→    51→
    52→    52→# Or manually copy a category if you only want one division
    53→    53→cp engineering/*.md ~/.claude/agents/
    54→    54→
    55→    55→# Then activate any agent in your Claude Code sessions:
    56→    56→# "Hey Claude, activate Frontend Developer mode and help me build a React component"
    57→    57→```
    58→    58→
    59→    59→### Option 3: Use as Reference
    60→    60→
    61→    61→Each agent file contains:
    62→    62→- Identity & personality traits
    63→    63→- Core mission & workflows
    64→    64→- Technical deliverables with code examples
    65→    65→- Success metrics & communication style
    66→    66→
    67→    67→Browse the agents below and copy/adapt the ones you need!
    68→    68→
    69→    69→### Option 4: Use with Other Tools (GitHub Copilot, Antigravity, Gemini CLI, OpenCode, OpenClaw, Cursor, Aider, Windsurf, Kimi Code, Codex, Osaurus, Hermes, Mistral Vibe)
    70→    70→
    71→    71→```bash
    72→    72→# Step 1 -- generate integration files for all supported tools
    73→    73→./scripts/convert.sh
    74→    74→
    75→    75→# Step 2 -- install interactively (auto-detects what you have installed)
    76→    76→./scripts/install.sh
    77→    77→
    78→    78→# Or target a specific tool directly
    79→    79→./scripts/install.sh --tool antigravity
    80→    80→./scripts/install.sh --tool gemini-cli
    81→    81→./scripts/install.sh --tool opencode
    82→    82→./scripts/install.sh --tool copilot
    83→    83→./scripts/install.sh --tool openclaw
    84→    84→./scripts/install.sh --tool cursor
    85→    85→./scripts/install.sh --tool aider
    86→    86→./scripts/install.sh --tool windsurf
    87→    87→./scripts/install.sh --tool kimi
    88→    88→./scripts/install.sh --tool codex
    89→    89→./scripts/install.sh --tool osaurus
    90→    90→./scripts/install.sh --tool hermes
    91→    91→./scripts/install.sh --tool vibe
    92→    92→```
    93→    93→
    94→    94→**Install only the teams you need** (not everyone wants every division):
    95→    95→
    96→    96→```bash
    97→    97→./scripts/install.sh                                    # interactive wizard: pick tools + teams
    98→    98→./scripts/install.sh --tool claude-code --division engineering,security
    99→    99→./scripts/install.sh --tool cursor --agent frontend-developer,ui-designer
   100→   100→./scripts/install.sh --list teams                       # see every team + agent count
   101→   101→./scripts/install.sh --tool opencode --division engineering --dry-run
   102→   102→```
   103→   103→
   104→   104→> **OpenCode note:** OpenCode's runtime currently registers only ~119 agents and silently drops the rest ([upstream bug](https://github.com/anomalyco/opencode/issues/27988)). Installing a subset with `--division` keeps you under that limit. The installer warns you when a selection would exceed it.
